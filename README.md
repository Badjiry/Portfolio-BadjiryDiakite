🛠️ Stack Technique
Ce portfolio a été conçu pour être rapide, moderne et entièrement responsive.

Frontend : HTML5, JavaScript (ES6+), Tailwind CSS (via CDN).

Design : Intégration de FontAwesome pour les icônes et utilisation de vidéos en arrière-plan pour une expérience immersive.


Outils : Git, GitHub LFS (pour la gestion des fichiers vidéos volumineux).

📂 Structure du Projet
Le projet est organisé de manière simple pour faciliter la navigation :

index.html : Page d'accueil avec présentation vidéo et accès rapide au CV.

Propos.html : Détails sur mes compétences techniques (Frontend, Backend, Outils) et mon parcours de formation.

Projets.html : Galerie de mes réalisations avec descriptions détaillées et liens vers les codes sources.

Contact.html : Informations pour me joindre (Email, Téléphone, LinkedIn).

Assets/ : Contient les images, les vidéos de démonstration et mon CV au format PDF/JPG.
